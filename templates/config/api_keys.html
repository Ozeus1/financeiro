Regras de segurança
  • A chave é exibida uma única vez — salve imediatamente para o agente de IA.
  • Gerar nova chave invalida a anterior imediatamente.
  • Cada chave dá acesso apenas aos dados do usuário dono da chave.
  • O admin não acessa dados financeiros — apenas gerencia as chaves de acesso.
  • Em caso de comprometimento, revogue imediatamente e gere nova chave.
